# SocialForge Roadmap

SocialForge ships with 17 skills. Here's what's planned. Contributions welcome — see [`CONTRIBUTING.md`](CONTRIBUTING.md).

## Shipped since launch
`profile-optimizer`, `offer-cta-architect`, `youtube-title-lab` — now in `skills/` (v1.1, 20 skills total).

## Next skills (help wanted)

- **`newsletter-growth`** — convert social attention into owned email audience; lead magnets and welcome sequences.
- **`ugc-brief`** — briefs for creators/UGC when scaling beyond yourself.
- **`comment-mining`** — turn your (and competitors') comment sections into a content-idea engine.
- **`ab-thumbnail-lab`** — structured thumbnail experiments and read-outs.

## Platform packs

Deeper platform-specific variants: `youtube-pack`, `tiktok-pack`, `linkedin-pack`, `x-pack`, `instagram-pack`.

## Integrations (design stage)

Hooks so the skills plug into automation:
- Webhook/JSON output modes for Make, Zapier, and n8n.
- A scheduling/queue adapter (draft → schedule → post).
- Optional pairing with a browsing/search tool so trend and analytics skills can pull live data.

## Concepts

- **Growth Dashboard** — a single view that runs `analytics-interpreter` + `performance-optimizer` on a batch of posts and returns the week's priorities.
- **Prompt packs** — ready-made input bundles per niche (fitness, SaaS, local business, personal brand).

Have an idea or a skill to contribute? Open an issue or PR.
