# Repurpose Master

One deliberate long asset contains 30-50 pieces of short-form content — most creators publish it once and waste 95% of the value. The unlock is atomization: every quotable line, mini-lesson, hot take, story, and Q&A is a standalone clip with its own hook. You are not slicing the video into equal chunks; you are hunting the moments that already spike, then rebuilding each as a self-contained piece with a hook that assumes zero context. One idea per clip is non-negotiable — a clip that tries to say two things gets swiped on both.

## 1. Mine the asset for atomic moments
1. Pass through the full asset and timestamp every candidate: a quotable line, a counterintuitive claim, a mini-framework, a personal story, a sharp Q&A exchange, a strong reaction or laugh.
2. Tag each by type — QUOTE, LESSON, HOT-TAKE, STORY, Q&A, DEMO — because type dictates format and platform.
3. Score for standalone strength: does it make sense with no setup? Does it provoke, teach, or surprise in under 3s? Kill anything that needs the surrounding 5 minutes.
4. Target 30+ from a 45-60 min asset. Dense assets yield more; don't force weak cuts to hit a number.

## 2. Spec each cut
1. Assign platform by fit: hot takes and quotes → TikTok/Reels/Shorts; frameworks and lessons → carousel + LinkedIn text; stories → Reels + Shorts; Q&A → Shorts + tweet.
2. Write a fresh hook per clip — the source's opening line is almost never the hook. The strongest sentence usually sits 10-20s into the moment; open there.
3. Set length, aspect ratio (9:16 vertical, 1:1 square, 16:9 native), and on-screen text keyword.
4. One asset feeds many formats: the same 40s moment becomes a vertical clip, a quote-card image, a carousel slide, and a text post — four pieces, one source.

## 3. Batch workflow
1. Mine once, spec all 30+ before editing anything — context-switching between watching and editing is the time sink.
2. Group by format and cut in batches (all vertical clips, then all quote cards, then all carousels).
3. Stagger the calendar: never publish two derivatives of the same moment on the same platform same day.
4. Track which derivatives outperform, then commission more long assets around those angles.

## 1 pillar → many derivatives map
```
PILLAR: <long asset>
 ├─ 8-12 vertical clips (TikTok / Reels / Shorts)
 ├─ 3-5 quote cards (Instagram / LinkedIn image)
 ├─ 2-3 carousels (Instagram / LinkedIn, 5-8 slides)
 ├─ 5-8 text posts / tweets (X / Threads / LinkedIn)
 ├─ 1 email newsletter section
 └─ 1 blog post / show notes (SEO)
```

## Platform variants
- **TikTok/Reels/Shorts**: 9:16, 20-40s, punchy hook, native captions.
- **LinkedIn/X**: text-first; lift the quote, add one line of framing; carousels crush for frameworks.
- **Instagram**: mix clips + quote cards + carousels; 4:5 for feed, 9:16 for reels.
- **Email/Blog**: use the lesson-type moments as the backbone; link back to the pillar.

## Rules
- One idea per clip. If a moment carries two, ship two clips.
- Rewrite the hook for every derivative — the source's first words are almost never the hook.
- Mine everything before you edit anything. Batch by format.
- A clip that needs context to land is not a clip — cut it or reframe it to stand alone.
- Never republish identical derivatives across platforms same-day; re-slice, don't cross-post raw.
- Chase the moment, not the number. 22 strong clips beat 40 with filler.
